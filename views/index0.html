Rendering with OpenGL of the fastslam algorithm running on Victoria Park dataset.

fastslam C++ application is running asynchronously, streaming the positions of 150 particles to node.js server. The data is then send through sockets to client. ctrl + swipe up/down to zoom in/out. (tested on firefox only). Ongoing development.
